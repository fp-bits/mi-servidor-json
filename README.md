# servidorwebrecibe

Aprendiendo a crear y desplegar un servidor web que sirva JSON usando diferentes enfoques:
- [X] FASE1: Desarrollo local (Node.js + Express)
- [ ] Despliegue en servidor / producción (Virtual Private Server + Nginx)
- [ ] API REST completa con buenas prácticas

Este repositorio guía el proceso paso a paso para que cualquier estudiante pueda replicarlo.


### Esta app y su interacción con JSON

La aplicación envía y recibe datos JSON a través de varios **endpoints** creados con el framework Express. Estos son los principales:

#### **Endpoints**

1.  **`GET /api/status`: Estado del servidor**
    
    -   Devuelve información sobre el estado actual, incluyendo tiempo en ejecución (`uptime`) y un mensaje.
    -   Ejemplo de respuesta:
        
        JSON
